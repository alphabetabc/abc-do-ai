# carousel-list（轮播列表垂直）

## 1. 概述

**名称**：轮播列表(垂直)（carousel-list）

**用途**：垂直轮播列表组件，支持自动轮播、选中高亮、自定义样式、多色配置。

**所属分类**：轮播 / 公告

**复杂度**：中

### 1.1 文件入口

| 文件 | 作用 |
| --- | --- |
| `index.jsx` | 主组件入口 |
| `index.less` | 样式 |
| `schema.ts` | 配置面板定义 |
| `dataModel.json` | 数据契约 |
| `oss-material.json` | 物料元信息 |

### 1.2 核心能力

- **自动轮播**：支持配置轮播时间间隔，自动切换选中项
- **选中高亮**：支持自定义选中项的背景、边框、阴影样式
- **多色配置**：支持为名称、数值、单位分别配置多色
- **分割线样式**：支持自定义分割线颜色、长度、虚线样式
- **交互派发**：支持派发选中行的 ID 和名称参数

### 1.3 适用场景

- 大屏展示垂直列表数据
- 需要自动轮播高亮的场景
- 需要多色区分的列表展示

## 3. Schema 结构（摘要）

Schema 分为 6 个主要面板：
- **功能**：轮播时间间隔
- **样式**：行宽度、行高度、图标尺寸、图标自定义填充色
- **分割线样式**：颜色、总长度、虚线长度、虚线密度
- **选中样式**：背景颜色、边框颜色、边框样式、边框宽度、阴影尺寸、阴影颜色
- **名称样式**：字体、字号、粗细、是否多色、字体颜色/自定义填充色
- **数值样式**：字体、字号、粗细、是否多色、字体颜色/自定义填充色
- **单位样式**：字体、字号、粗细、是否多色、字体颜色/自定义填充色

交互面板：
- **行高亮事件**：派发参数（行数据ID、行数据名称）

详见 [schema.md](./schema.md)。

## 4. 组件逻辑（摘要）

主组件 `CarouselList` 渲染流程：
1. **数据解析**：从 dataSource 读取 id、name、value、unit 字段
2. **样式构建**：根据配置构建行样式、分割线样式、选中样式、字体样式
3. **轮播逻辑**：使用 setInterval 自动切换 currentIndex
4. **选中状态**：根据 current 状态判断是否应用选中样式
5. **交互派发**：选中项变化时派发 rowHighlightId 和 rowHighlightName 参数

详见 [component-logic.md](./component-logic.md)。

## 5. 数据格式（摘要）

dataModel.json 定义了 4 个指标字段：
- `id`（ID）
- `name`（名称）
- `value`（值）
- `unit`（单位）

数据流向：
```
外部数据源 → dataModel.json → dataSource → index.jsx → 渲染列表
```

详见 [data-model.md](./data-model.md)。

## 7. 注意事项

详见 [gotchas.md](./gotchas.md)。

主要踩坑：
- 轮播时间间隔小于 1 时不生效
- 多色配置使用 ColorGroup 组件
- 分割线使用 backgroundImage 实现虚线效果
- 选中状态使用 name 字段判断
