# 样式

样式集中在 [index.css](file:///e:/oss-fe-git/phoenix/oss-visual-gd-emergency-support-next/apps/main/app/components/left/emergency-resources/index.css)。

## 关键 class

| class | 作用 |
| --- | --- |
| `.emergency-resources-content` | 主图表容器 |
| `.emergency-resources-extra-child` | 右上页签与资源清单按钮 |
| `.emergency-resources-type` | 页签基础样式 |
| `.emergency-resources-type-selected` | 页签选中样式 |
| `.emergency-resources-detail-modal` | 详情弹窗整体样式 |
| `.emergency-resources-detail-content` | 弹窗内容布局 |
| `.emergency-resources-detail-region` | 队伍详情区域选择 |
| `.emergency-resources-detail-stats` | 队伍详情类型统计 |
| `.emergency-resources-detail-table` | 详情表格样式 |

## 注意

项目中的表格样式使用 `.fedx-ui-*` 类名覆盖 Ant Design/FedX UI 表格，修改时要确认实际 DOM 类名是否仍一致。
