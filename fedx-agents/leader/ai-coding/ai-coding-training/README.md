# AI Coding 培训材料

全员研发统一认知：**文档驱动 AI 写代码**。

## 怎么读（唯一顺序）

按下列标题顺序读，不要跳着从提示词开始。现场讲解按同样顺序依次讲即可。

| 顺序 | 文档 | 解决什么问题 |
|------|------|----------------|
| 1 | [为什么做与思维转变](./mindset.md) | 危急存亡、会提问；个人/团队同一套尺子；三个方向 |
| 2 | [新项目开发流程](./new-project.md) | **主路径**：反向提问 · 五件套/design/research · Superpowers 等工具 |
| 3 | [历史项目维护](./legacy-project.md) | 仓库级 S1–S5 改造 · as-is 取证 P1–P6 · 单次 as-is→to-be |
| 4 | [提示词与自检](./cheatsheet.md) | 先选路 · P/S 速查 · 新/历史提示词按阶段 · 禁止项与自检 |

浏览器审阅（同序）：[html/index.html](./html/index.html)

```text
为什么做与思维转变
        ↓
新项目开发流程（脊梁）
        ↓

        ↓
提示词与自检
```
